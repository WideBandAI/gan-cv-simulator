## GaN C-V特性シミュレーションプログラム 仕様書

### 1. 概要

本プログラムは、Rust言語を用いて、金属-絶縁膜-半導体（MIS）構造を持つGaNキャパシタの静電容量-電圧（C-V）特性をシミュレーションすることを目的とする。
ユーザーはデバイスの物理パラメータとシミュレーション条件を入力し、結果としてC-Vカーブのデータを得ることができる。
実装はコマンドラインインターフェース（CLI）アプリケーションとして行う。

### 2. 機能要件

#### 2.1. 入力パラメータ

シミュレーションの挙動を制御するため、以下のパラメータを入力として受け付ける。入力形式は設定ファイル（例: TOML, JSON）またはコマンドライン引数を想定する。

**2.1.1. デバイス構造 (`structure.rs`で定義)**

*   **絶縁膜 (Insulator):**
    *   厚さ (nm)
    *   比誘電率
*   **GaN半導体 (Semiconductor):**
    *   層の厚さ (μm)
    *   ドーピングタイプ (n型 / p型)
    *   ドーピング濃度 (cm⁻³)
    *   バンドギャップ (eV)
    *   比誘電率
    *   電子親和力 (eV)
*   **金属ゲート (Gate Metal):**
    *   仕事関数 (eV)
*   **(オプション) 界面:**
    *   界面準位密度 (cm⁻²eV⁻¹)

**2.1.2. シミュレーション条件**

*   **電圧掃引:**
    *   開始電圧 (V)
    *   終了電圧 (V)
    *   電圧ステップ (V)
*   **物理条件:**
    *   温度 (K)
*   **(オプション) AC解析:**
    *   測定周波数 (Hz)

#### 2.2. シミュレーションプロセス

1.  **デバイスの離散化:**
    *   1次元のMIS構造を仮定し、メッシュ（格子点）に分割する。
2.  **DCバイアス計算:**
    *   指定された各ゲート電圧において、半導体内の物理状態を計算する。
    *   具体的には、ポアソン方程式とキャリアの連続の方程式を自己無撞着に解き、電位分布、キャリア密度分布を求める。
3.  **静電容量計算:**
    *   各DCバイアス点において、ゲート電圧を微小に変化させた際の電荷の変化量 (ΔQ) を計算し、静電容量 C = dQ/dV を算出する。

#### 2.3. 出力

*   計算結果をCSV形式のファイルに出力する。
    *   列: `Voltage (V)`, `Capacitance (F/cm^2)`
*   (オプション) 計算結果をプロットし、画像ファイル（例: PNG）として保存する。

### 3. アーキテクチャ

プログラムを機能ごとにモジュール分割し、保守性と拡張性を高める。

*   `main.rs`:
    *   コマンドライン引数の解析（`clap` クレートを利用）。
    *   入力ファイルの読み込み。
    *   シミュレーションの全体的な制御フローを管理。
    *   各モジュールを呼び出し、計算を実行させる。
*   `src/structure.rs`:
    *   デバイスの物理パラメータや構造を定義する構造体（`struct`）を配置する。
    *   例: `DeviceParameters`, `Insulator`, `Semiconductor` など。
*   `src/solver.rs`:
    *   1Dポアソン方程式などを解く数値計算のコアロジックを実装する。
    *   有限差分法とNewton法などの反復解法を用いる。
    *   数値計算ライブラリ `ndarray` や `nalgebra` の利用を推奨。
*   `src/cv.rs`:
    *   `solver`モジュールを利用して、指定された電圧範囲で掃引計算を行い、C-V特性を算出する。
*   `src/output.rs`:
    *   計算結果をCSVファイルや画像ファイルに出力する機能を実装する。
    *   グラフ描画には `plotters` クレートの利用を推奨。

### 4. 開発マイルストーン

1.  **フェーズ1: 基本構造の構築**
    *   `src/structure.rs` に、シミュレーションに必要な物理パラメータを保持する構造体を定義する。
    *   `main.rs` にて、`clap` を用いて基本的なコマンドライン引数（電圧範囲など）をパースする処理を実装する。
2.  **フェーズ2: DC解析の実装**
    *   `src/solver.rs` に、有限差分法を用いて1次元ポアソン方程式を解く機能を実装する。まずは熱平衡状態（ゲート電圧 0V）での電位分布を計算できるようにする。
3.  **フェーズ3: C-V計算と結果出力**
    *   `src/cv.rs` で、電圧を掃引しながら各バイアスポイントでの電荷量を計算し、静電容量を算出するロジックを実装する（準静的C-V）。
    *   `src/output.rs` で、計算結果（電圧, 容量）をCSVファイルに出力する機能を実装する。
4.  **フェーズ4: (オプション) 機能拡張**
    *   不完全イオン化、界面準位などの高度な物理モデルを `solver` に組み込む。
    *   `plotters` を利用してC-Vカーブを画像として出力する機能を追加する。
    *   GaN特有の分極効果（自発分極・ピエゾ分極）をモデルに組み込む。

### 5. 非機能要件

*   **言語:** Rust (Stable)
*   **パフォーマンス:** `ndarray` などの効率的な数値計算ライブラリを活用し、デスクトップPCで数分以内に計算が完了することを目指す。
*   **可読性:** Rustの標準的なフォーマット（`rustfmt`）に従い、適切なコメントを付与する。
*   **テスト:** `solver` の単体テストを作成し、基本的な物理法則（例: 電荷中性条件）が満たされていることを確認する。
